# 投资组合

非正态分布的模型和正态分布的模型

在量化投资与资产配置中，传统的金融理论通常假设资产收益率符合**正态分布（Normal Distribution）**。然而，现实市场往往存在“尖峰厚尾”和“非对称”的特征，因此衍生出了许多非正态分布（Non-normal Distribution）的模型。

为了让你清晰地掌握这一体系，我将这些模型分为**正态分布基准模型**和**非正态分布改良模型**两大类，并细分为不同的应用维度：

## 一、 正态分布模型（经典金融学基准）

这类模型核心假设是资产收益率服从正态分布，或者投资者只关心**均值（期望收益率）**和**方差（风险）**。

### 1. 资产配置与投资组合构建

- **马科维茨均值-方差模型（Mean-Variance Optimization, MVO）：** 现代投资组合理论（MPT）的基石。仅使用期望收益率（均值）和协方差矩阵（方差）来寻找有效前沿。
    
- **Black-Litterman 模型：** 虽然引入了主观观点并结合了市场均衡，但其底层数学框架（先验分布和条件分布）依然完全基于**多元正态分布**。
    

### 2. 资产定价与风险度量

- **资本资产定价模型（CAPM）：** 假设资产收益率由系统性风险 $\beta$ 决定，隐含了正态分布或二次效用函数的假设。
    
- **传统风险价值（Parametric VaR / Delta-Normal VaR）：** 在计算风险价值（Value at Risk）时，直接使用正态分布的 $Z$ 分数（如 $1.65$ 或 $2.33$）来估计最大潜在损失。
    

## 二、 非正态分布模型（现代量化与波动率模型）

现实市场中，资产收益率经常出现极端行情的“厚尾”现象。为了捕捉这些特征，衍生出了以下几类非正态模型：

### 1. 基于替代概率分布的模型（修正统计假设）

这类模型直接抛弃正态分布，换用能捕捉“尖峰厚尾”或“偏度”的统计分布。

- **学生 $t$ 分布模型（Student's t-Distribution）：** 拥有比正态分布更厚胖的尾部，自由度 $v$ 越小，尾部越厚，非常适合模拟金融资产的常态极端波动。
    
- **偏态 $t$ 分布与广义误差分布（GED）：** 不仅能捕捉厚尾，还能捕捉左偏（暴跌概率更高）或右偏的非对称特征。
    
- **稳定帕累托分布（Stable Paretian Distribution）：** 专门用于模拟具有“无限方差”特征的极端巨幅波动（分形市场假说）。
    

### 2. 动态波动率与时变模型（修正独立同分布假设）

正态模型通常假设波动率是恒定的，而这类模型认为波动率会随时间变化，从而在整体上呈现出非正态的“波动率聚集”效应。

- **ARCH / GARCH 系列模型：** 自动调节条件方差。即使残差项假设为正态，由于波动率的时变性，最终得到的收益率边缘分布也是非正态（厚尾）的。
    
- **随机波动率模型（Stochastic Volatility, SV）：** 将波动率本身视为一个随机过程，常用于衍生品高级定价。
    

### 3. 资产配置与风险管理的非正态改良（进阶模型）

- **CVaR（条件风险价值）与期望尾部损失（Expected Shortfall, ES）：** 传统的 VaR 在非正态下无法量化极端尾部究竟有多大损失。CVaR 专门计算超越 VaR 阈值后的平均损失，对非正态分布极其友好。
    
- **基于高阶矩的投资组合优化：** 在均值-方差的基础上，引入**偏度（Skewness）**和**峰度（Kurtosis）**。优化目标变为：最大化收益和偏度，最小化方差和峰度。
    
- **风险平价模型（Risk Parity / HRP）：** * **传统风险平价**依赖协方差（正态假设）。
    
    - **改良版风险平价 / 分层风险平价（HRP）：** 可以利用非线性相关性（如互信息、Distanec Correlation）或基于历史模拟/蒙特卡洛（无需假设分布）来分配风险权重，从而免疫非正态特征。
        

### 4. 依赖关系非正态模型（捕捉协同暴跌）

- **Copula 模型（如 Clayton Copula, Gumbel Copula）：** 传统的线性相关系数（Pearson $r$）假设资产间是多元正态关系。而 Copula 能够把资产的个体分布和它们之间的尾部相关性（Tail Dependence）分开建模。例如，捕捉“平时不相关，股灾时一起暴跌”的非对称非正态相关性。
    

## 💡 总结对比表

|**维度**|**正态分布模型 (经典)**|**非正态分布模型 (现代量化)**|
|---|---|---|
|**核心假设**|收益率服从 $N(\mu, \sigma^2)$，无尖峰厚尾|收益率有偏度、厚尾、波动率聚集|
|**代表模型**|Markowitz MVO, CAPM, Black-Litterman, Delta-Normal VaR|GARCH, Copula, 均值-方差-偏度-峰度优化, HRP|
|**风险度量**|方差 (Variance)、标准差、传统 VaR|CVaR / 期望损失 (ES)、下行偏差 (Downside Deviation)|
|**相关性度量**|线性相关系数 (Pearson $\rho$)|尾部相关性 (Tail Dependence)、互信息|
|**优缺点**|📊 **优：** 有解析解，计算极快，理论完美。<br><br>  <br><br>❌ **缺：** 低估极端风险（黑天鹅）。|🎯 **优：** 贴合市场实际，防范尾部风险。<br><br>  <br><br>❌ **缺：** 计算复杂，容易过拟合，参数敏感。|

在实际的 Financial Engineering 业务中，通常会以正态模型作为**基准快速筛选**，但在高频交易、衍生品定价以及极端的尾部风险压力测试（Stress Testing）中，则必须切换到非正态模型（如 GARCH + Student-t 或 Copula）来确保策略的鲁棒性。
