## 什么是 Big O 表示法？

Big O 表示法（大 O 表示法）是一种数学符号，用来描述算法在“最坏情况下”的增长趋势。

Big O (渐进上界)：它不关注精确的执行次数，而是关注随着输入规模 nn 的增加，运行时间的增长量级。例如，如果操作次数是 2n+52n+5，在 Big O 看来，它就是 O(n)O(n)，因为常数项和系数在极大的 nn 面前影响很小。

## O(1) - 常数时间 (Constant Time)

无论数据量 n 有多大，算法执行的时间始终保持不变。

## O(log n) - 对数时间 (Logarithmic Time)

每执行一步，问题的规模都会“折半”。这是一种非常高效的算法

## O(n) - 线性时间 (Linear Time)

运行时间与输入规模 nn 成正比。如果数据量增加 10 倍，时间也增加 10 倍

```
- 最快：O(1)O(1) —— 瞬间完成，与数量无关。
- 极快：O(log⁡n)O(logn) —— 面对海量数据也能游刃有余。
- 普通：O(n)O(n) —— 中规中矩，数据翻倍，时间翻倍。
```

> 可以用 python 计算 O 的时间复杂度 ?

```python
import matplotlib.pyplot as plt
import numpy as np

# --- 1. 准备数据 ---
# 定义输入规模 n，从 1 到 100
n = np.arange(1, 101)

# 计算不同的复杂度数值
o_1 = np.ones_like(n)              # O(1)
o_log_n = np.log2(n)               # O(log n) - 使用底数为 2 的对数
o_n = n                            # O(n)
o_n_log_n = n * np.log2(n)         # O(n log n)
o_n_squared = n**2                 # O(n^2)

# --- 2. 绘制图表 ---
plt.figure(figsize=(10, 6))

plt.plot(n, o_1, label='O(1) 常数时间', linewidth=2)
plt.plot(n, o_log_n, label='O(log n) 对数时间', linewidth=2)
plt.plot(n, o_n, label='O(n) 线性时间', linewidth=2)
plt.plot(n, o_n_log_n, label='O(n log n) 线性对数时间', linewidth=2)
plt.plot(n, o_n_squared, label='O(n^2) 平方时间', linewidth=2)

# --- 3. 图表美化与设置 ---
plt.yscale('log') # 关键点：使用对数轴，方便在同一张图里观察量级差异
plt.title('Big O 复杂度增长趋势对比', fontsize=14)
plt.xlabel('输入规模 (n)', fontsize=12)
plt.ylabel('操作次数 (对数尺度)', fontsize=12)
plt.legend()
plt.grid(True, which="both", ls="-", alpha=0.5)

# 保存并显示
plt.tight_layout()
plt.show()
```
