# IS NULL Syntax

```sql
SELECT column_names
FROM table_name
WHERE column_name IS NULL; 
```

# IS NOT NULL Syntax

```sql
SELECT column_names
FROM table_name
WHERE column_name IS NOT NULL; 
```
