# 交易策略

- **金叉（买入信号）**：短期均线从下向上穿过长期均线。意味着短期价格动能走强，预示可能开启上涨趋势。
    
- **死叉（卖出信号）**：短期均线从上向下穿过长期均线。意味着短期价格动能转弱，预示可能开启下跌趋势。

# 代码

```python
import yfinance as yf
import pandas as pd
import matplotlib.pyplot as plt

# 1. 下载数据
df = yf.download('AAPL', start='2024-01-01', end='2025-01-01')

# 2. 计算均线 (SMA)
# 短期均线：5日线；长期均线：20日线
df['Short_MA'] = df['Close'].rolling(window=5).mean()
df['Long_MA'] = df['Close'].rolling(window=20).mean()

# 3. 生成信号
# 当短线 > 长线，标记为 1 (持仓)；否则标记为 0 (空仓)
df['Signal'] = 0.0
df['Signal'] = (df['Short_MA'] > df['Long_MA']).astype(float)

# 计算交叉点：Signal的变化即为买卖点
# 1表示金叉，-1表示死叉
df['Position'] = df['Signal'].diff()

# 4. 绘图
plt.figure(figsize=(14, 7))

# 绘制收盘价和均线
plt.plot(df['Close'], label='Close Price', alpha=0.5, color='gray')
plt.plot(df['Short_MA'], label='5-Day MA', color='blue', lw=1.5)
plt.plot(df['Long_MA'], label='20-Day MA', color='orange', lw=1.5)

# 标注金叉 (买入信号)
plt.plot(df[df['Position'] == 1].index, 
         df['Short_MA'][df['Position'] == 1], 
         '^', markersize=10, color='red', label='Golden Cross (Buy)')

# 标注死叉 (卖出信号)
plt.plot(df[df['Position'] == -1].index, 
         df['Short_MA'][df['Position'] == -1], 
         'v', markersize=10, color='green', label='Death Cross (Sell)')

plt.title('AAPL Golden Cross & Death Cross Strategy')
plt.legend()
plt.grid(True)
plt.show()
```
