将“金融工程基础”与“美股实战”结合，并针对上班族和学生的节奏，这30天的教学大纲可以分为**四个核心阶段**。

教学目标是从“金融直觉”转向“工程逻辑”，即不仅知道买什么，还知道为什么要买，以及如何用代码验证。

---

## 第一阶段：市场架构与环境搭建（第1-7天）

**核心目标：** 理解美股规则，准备好生产力工具。

- **Day 1-2：美股微观结构。** 详解 T+0、融资融券、盘前盘后。对比纽交所与纳斯达克的订单撮合机制。
    
- **Day 3-4：金融工程环境。** 安装 Python 核心库（Pandas, Numpy, Matplotlib）。配置 VS Code 或 Jupyter Notebook。
    
- **Day 5-7：数据获取实战。** 学习调用 yfinance 或 Alpha Vantage API 获取分钟级/日级行情，并清洗无效数据。
    

---

## 第二阶段：定量分析基础（第8-15天）

**核心目标：** 将股价变化转化为数学指标。

- **Day 8-10：收益率与正态分布。** 计算对数收益率。通过直方图观察美股的“胖尾效应”（Fat Tail），理解极端风险。
    
- **Day 11-12：移动平均与动量。** 实现 SMA、EMA 指标。理解技术指标背后的信号平滑逻辑。
    
- **Day 13-15：风险度量（Risk Metrics）。** 编写函数计算最大回撤（MDD）、夏普比率（Sharpe Ratio）。
    

---

## 第三阶段：策略构建与回测系统（第16-23天）

**核心目标：** 建立不带情绪的交易逻辑。

- **Day 16-17：择时策略开发。** 编写第一个“金叉死叉”或“布林带回归”策略。
    
- **Day 18-20：Backtrader 回测框架。** 学习如何将历史数据喂给策略，模拟真实交易中的滑点（Slippage）和手续费。
    
- **Day 21-23：参数优化。** 学习使用网格搜索（Grid Search）寻找最佳参数，同时学习如何识别“过拟合”陷阱。
    

---

## 第四阶段：进阶实战与投资组合（第24-30天）

**核心目标：** 风险分散与实盘模拟准备。

- **Day 24-25：多因子模型入门。** 尝试同时根据“波动率”和“动量”两个维度筛选标普 500 成分股。
    
- **Day 26-27：仓位管理艺术。** 学习凯利公式（Kelly Criterion）。计算在 2% 风险敞口下，单笔交易应买入多少股。
    
- **Day 28-29：Paper Trading。** 注册 IBKR 或富途等券商的模拟盘，将 Python 逻辑对接模拟账户。
    
- **Day 30：复盘与管理。** 建立 Obsidian 交易日志，记录每笔策略交易的逻辑一致性而非单纯盈亏。
