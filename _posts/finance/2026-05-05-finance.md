衡量投资组合的评价方法

它不仅关注投资组合赚了多少钱，更核心的目的是**区分管理能力（Skill）与运气（Luck）**

业绩评价的技术旨在回答：投资组合的超额收益是来自于承担了更高的风险，还是来自于经理人的卓越决策

业绩评价与归因分析是一套**诊断工具**。它利用标准化的**评价技术**（如第 25.1 节所述）来衡量绩效，并通过**分解技术**和**多因子模型**来透视收益背后的结构，从而帮助投资者判断管理人的策略是否具有可持续性。

如何计算核心评价指标（夏普、特雷诺、詹森 Alpha）以及如何通过回归分析进行**收益归因**。

```python
import yfinance as yf
import pandas as pd
import numpy as np
import statsmodels.api as sm

def get_real_data(ticker, benchmark, start_date, end_date):
    # 获取收盘价并计算日收益率
    data = yf.download([ticker, benchmark], start=start_date, end=end_date)['Close']
    returns = data.pct_change().dropna()
    return returns[ticker], returns[benchmark]

def analyze_actual_performance(ticker="AAPL", benchmark="SPY"):
    # 1. 准备数据 (过去一年)
    port_ret, mkt_ret = get_real_data(ticker, benchmark, "2023-01-01", "2024-01-01")
    rf_daily = 0.04 / 252  # 假设年化无风险利率 4%
    
    excess_port = port_ret - rf_daily # numpy list
    excess_mkt = mkt_ret - rf_daily

    # 2. 业绩评价 (CAPM 回归)
    X = sm.add_constant(excess_mkt)
    model = sm.OLS(excess_port, X).fit()
    
    alpha_daily = model.params['const']
    beta = model.params[benchmark]
    
    # 指标计算
    sharpe = (excess_port.mean() / excess_port.std()) * np.sqrt(252)
    treynor = (excess_port.mean() * 252) / beta
    jensen_alpha = alpha_daily * 252
    
    # 3. 输出结果
    print(f"--- {ticker} 业绩评价报告 ---")
    print(f"年化夏普比率: {sharpe:.4f}")
    print(f"年化特雷诺比率: {treynor:.4f}")
    print(f"年化詹森 Alpha: {jensen_alpha:.4%}")
    print(f"系统性风险 Beta: {beta:.4f}")
    print(f"R-squared (基准解释力): {model.rsquared:.4f}")

analyze_actual_performance()
```
