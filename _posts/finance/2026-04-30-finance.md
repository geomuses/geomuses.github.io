肥尾效应（Fat-tailed distribution）是指极端事件（如股市崩盘或暴涨）发生的概率远高于正态分布（高斯分布）的预测

```python
import yfinance as yf
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

# 1. 获取数据 (以标普500 ETF - SPY 为例)
ticker = "SPY"
data = yf.download(ticker, start="2014-01-01", end="2024-01-01")

# 2. 计算每日对数收益率
# 使用对数收益率能更好地体现累积效应
data['Returns'] = np.log(data['Adj Close'] / data['Adj Close'].shift(1))
returns = data['Returns'].dropna()

# 3. 计算正态分布的参数 (均值和标准差)
mu, std = returns.mean(), returns.std()

# 4. 绘图
plt.figure(figsize=(12, 7))

# 绘制实际收益率的直方图
plt.hist(returns, bins=100, density=True, alpha=0.6, color='skyblue', label='实际收益率 (SPY)')

# 绘制理论正态分布曲线
xmin, xmax = plt.xlim()
x = np.linspace(xmin, xmax, 100)
p = norm.pdf(x, mu, std)
plt.plot(x, p, 'r', linewidth=2, label='理论正态分布')

# 5. 图表修饰
plt.title(f'{ticker} 收益率分布 vs 正态分布 (肥尾效应可视化)', fontsize=15)
plt.xlabel('每日收益率')
plt.ylabel('频率')
plt.legend()
plt.grid(axis='y', alpha=0.3)

# 标注肥尾区域
plt.annotate('左侧肥尾 (黑天鹅事件)', xy=(returns.min(), 0.05), xytext=(returns.min()*0.8, 10),
             arrowprops=dict(facecolor='black', shrink=0.05), fontsize=10)

plt.show()

# 6. 打印统计量验证
print(f"峰度 (Kurtosis): {returns.kurtosis():.2f}") 
print("注：正态分布的峰度为 0。如果峰度 > 0，则存在肥尾效应。")
```

会观察到以下三个关键特征：

1. **尖峰 (Leptokurtic):** 实际收益率在均值附近（中间部分）比正态分布更“瘦高”，这意味着市场在大部分时间内波动非常小。
    
2. **肥尾 (Fat Tails):** 在图表的左右两侧极端区域，蓝色的直方图往往会高出红色的正态分布曲线。这说明**极端涨跌的频率比数学模型预测的要频繁得多**。
    
3. **峰度 (Kurtosis):**
    
    - 正态分布的超额峰度为 0。
        
    - 金融资产（如股票）的峰度通常远大于 0（例如 SPY 通常在 10 以上），这在数学上证明了肥尾的存在。os : 13.96
        
# 为什么这很重要？

传统的风险管理模型（如 VaR）如果假设市场是正态分布的，就会严重低估极端风险。**肥尾效应告诉我们：在金融市场中，“万一”发生的概率其实比你想象的要大。**
