# 什么是量化金融?

**量化金融 (Quantitative Finance)** 是一门结合了**数学模型、统计学、计算机科学**以及**金融理论**的交叉学科。其核心目标是通过数值计算和数据驱动的方法，对金融市场进行定价、风险管理以及寻找交易机会。

简单来说，传统的金融分析可能依赖于宏观判断或公司基本面研究，而量化金融则依赖于**算法和代码**，试图在海量数据中捕捉人类肉眼难以发现的规律

# 虚拟环境

Python 虚拟环境是一个隔离的 Python 运行环境，让你在同一台电脑上安装和使用多个项目所需的不同版本的 Python 包，而不会相互干扰。

使用 venv 创建和管理虚拟环境

以下以 venv 为例：

创建虚拟环境

```bash
python -m venv env
```

env 是虚拟环境的目录名，可以自定义

会创建一个名为 env/ 的文件夹，里面包含独立的 Python 解释器和 site-packages 包目录

激活虚拟环境

Windows：

```bash
.\env\Scripts\activate
```

macOS / Linux：

```bash
source env/bin/activate
```

退出虚拟环境

```bash
deactivate
```
