# 为什么 CVaR 比 VaR 更重要？

在做决策时，单纯看 VaR 有一个致命弱点：**它只告诉你门槛，不告诉你门槛后的悬崖有多深**。

- **VaR** 说：“你有 5% 的概率损失超过 10 万。”
    
- **CVaR** 说：“在那 5% 的极端情况下，你的**平均损失**是 25 万。”
    
**预期不足 (Conditional Value at Risk, CVaR)**，也称为 **Expected Shortfall (ES)**，是比 VaR 更进阶的风险指标。

如果说 VaR 问的是“最坏的 5% 情况从哪里开始？”，那么 CVaR 问的就是“**如果真的掉进了那最坏的 5% 里面，平均会亏多少？**”

# CVaR 与 VaR 的直观对比

- **VaR (95%)**：是一个“阈值”。它告诉你，在 95% 的置信水平下，你的损失**不会超过**这个值。
    
- **CVaR (95%)**：是一个“均值”。它计算的是所有超过 VaR 阈值的极端损失案例的平均值。它能捕捉到 VaR 忽略的“尾部风险”。
    
# 代码

```python
import yfinance as yf
import numpy as np
import pandas as pd

# 1. 获取 AAPL 数据 (保持与之前一致)
data = yf.download('AAPL', start='2022-01-01', end='2024-01-01')
returns = np.log(data['Adj Close'] / data['Adj Close'].shift(1)).dropna()

# 2. 设置参数
initial_investment = 1000000
conf_level = 0.95

# ---------------------------------------------------------
# 方法一：历史模拟法 CVaR (最常用且直观)
# ---------------------------------------------------------
# 先计算历史 VaR 阈值
var_threshold = np.percentile(returns, (1 - conf_level) * 100)

# 找出所有比 VaR 还惨的收益率 (尾部数据)
tail_losses = returns[returns <= var_threshold]

# 计算这些极端损失的平均值
cvar_historical = initial_investment * tail_losses.mean()

print(f"AAPL 历史 VaR (95%): ${abs(initial_investment * var_threshold):,.2f}")
print(f"AAPL 历史 CVaR (95%): ${abs(cvar_historical):,.2f}")

# ---------------------------------------------------------
# 方法二：参数法 CVaR (基于正态分布公式)
# ---------------------------------------------------------
from scipy.stats import norm

mu = returns.mean()
sigma = returns.std()
alpha = 1 - conf_level

# 正态分布下的 CVaR 公式: μ - σ * (φ(Z_alpha) / alpha)
# 其中 φ 是标准正态分布的概率密度函数 (PDF)
z_score = norm.ppf(alpha)
pdf_value = norm.pdf(z_score)

cvar_parametric = initial_investment * (mu - sigma * (pdf_value / alpha))

print(f"参数法 CVaR (95%): ${abs(cvar_parametric):,.2f}")
```

# 为什么金融监管更青睐 CVaR?

在 2008 年金融危机后，巴塞尔协议 III 等监管框架开始转向 CVaR，主要原因如下：

1. **次可加性 (Sub-additivity)**：这是 VaR 的致命缺陷。VaR 可能会出现“组合后的风险反而大于各部分风险之和”的逻辑错误。而 CVaR 在数学上是相容的，鼓励通过分散投资来降低风险。
    
2. **尾部盲区**：VaR 只看门槛，不看门后的深渊。如果两支基金的 VaR 相同，但基金 A 在极端情况下亏损 10%，基金 B 亏损 90%，VaR 无法分辨它们，但 CVaR 会明确指出基金 B 更危险。
