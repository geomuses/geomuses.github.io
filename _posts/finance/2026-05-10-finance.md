**价格走势图（反映趋势）**和**收益率直方图（反映波动分布）**

```python
import yfinance as yf
import matplotlib.pyplot as plt
import pandas as pd

# 1. 获取数据
ticker = "NVDA"  # 你可以换成任何代码，如 "AAPL" 或 "TSLA"
data = yf.download(ticker, start="2023-01-01", end="2024-05-20")

# 计算每日收益率 (用于直方图)
data['Daily Return'] = data['Adj Close'].pct_change()

# 2. 创建画布 (包含两个子图)
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(12, 10))
plt.subplots_adjust(hspace=0.4) # 调整子图间距

# --- 图表 1: 价格走势图 ---
ax1.plot(data.index, data['Adj Close'], color='#1f77b4', linewidth=2)
ax1.set_title(f'{ticker} Price Trend (Adjusted Close)', fontsize=14)
ax1.set_ylabel('Price (USD)')
ax1.grid(True, linestyle='--', alpha=0.7)

# --- 图表 2: 收益率直方图 ---
# 掉落缺失值 (第一天没有收益率)
returns = data['Daily Return'].dropna()
ax2.hist(returns, bins=50, color='#2ca02c', edgecolor='black', alpha=0.7)
ax2.set_title(f'{ticker} Daily Returns Histogram', fontsize=14)
ax2.set_xlabel('Daily Return Percentage')
ax2.set_ylabel('Frequency')
ax2.grid(axis='y', linestyle='--', alpha=0.7)

# 打印简单的统计数据
print(f"数据抓取完成！分析对象: {ticker}")
print(f"平均收益率: {returns.mean():.4f}")
print(f"收益率标准差 (波动率): {returns.std():.4f}")

plt.show()
```
