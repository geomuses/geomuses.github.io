`EdgarTools` 是目前获取美股 SEC（美国证券交易委员会）数据最优雅的工具之一。它最大的优势在于**将原始的 XBRL/XML 数据直接转换为结构化的 Python 对象**，让你能像操作本地数据一样分析财报。

# 安装与身份设置

SEC 要求所有访问 EDGAR 系统的请求必须包含身份标识（User-Agent），否则会被拦截。

```bash
pip install edgartools
```

```python
from edgar import *
import polars as pl
# 必须设置身份：姓名 <邮箱>
set_identity("Your Name your.email@example.com")
```

# 快速获取公司财务报表

这是你最关心的部分。`EdgarTools` 能自动对齐不同年份的会计科目。

```python
# 获取公司对象（支持 Ticker 或 CIK）
company = Company("AAPL")

# 获取最新的财务数据对象
financials = company.get_financials()

# 获取三大报表 (自动返回 Pandas DataFrame)
income_stmt = financials.income_statement()  # 利润表
balance_sheet = financials.balance_sheet()   # 资产负债表
cash_flow = financials.cash_flow_statement() # 现金流量表

# 先转为 Pandas，再转为 Polars
df_pl = pl.from_pandas(income_stmt.to_dataframe(), include_index=True)
print(df_pl)
```

# 筛选特定类型的申报文件 (Filings)

下载历史 10-K（年报）或 10-Q（季报）进行回测。

```python
# 获取最新的 10 份 10-K 年报
ten_ks = company.get_filings(form="10-K").head(10)

# 获取其中最新的一份
latest_filling = ten_ks.latest()

# 获取该报表的原文（HTML）或解析后的对象
# doc = latest_filling.obj() 
```

# 提取文本内容（如风险因素）

如果你想做情感分析或 NLP 挖掘，`EdgarTools` 可以直接提取 10-K 中的特定章节。

```python
# 提取 10-K 中的 "Item 1A: Risk Factors"（风险因素）
filing = company.get_filings(form="10-K").latest()
risk_factors = filing.obj()["Item 1A"]
print(risk_factors[:500]) # 打印前 500 个字符
```

# 小贴士

- **缓存机制**：`EdgarTools` 会在本地缓存下载过的文件，避免重复请求 SEC 导致被封 IP。
    
- **关于 13F**：如果你想研究桥水或巴菲特的持仓，可以使用 `get_filings(form="13F-HR")`，它能直接解析出完整的持仓列表数据框。
