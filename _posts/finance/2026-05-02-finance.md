**在险价值（Value at Risk, VaR）**被定义为在有效机会集中选择投资组合的一种重要工具，它用于衡量在给定的置信水平下，投资组合在未来特定时间内可能面临的最大潜在损失。

计算 VaR 通常有三种主流方法

**参数法（方差-协方差法）**、**历史模拟法**和**蒙特卡洛模拟法**

以下是实现这三种方法的 Python 代码示例：

```python
import yfinance as yf
import numpy as np
import pandas as pd
from scipy.stats import norm

# 1. 下载 AAPL 过去两年的历史数据
df = yf.download('AAPL', start='2022-01-01', end='2024-01-01')

# 2. 计算日对数收益率 (ln(Pt / Pt-1))
df['Log_Ret'] = np.log(df['Adj Close'] / df['Adj Close'].shift(1))
returns = df['Log_Ret'].dropna()

# 3. 设置核心参数
initial_investment = 1000000  # 100万美金
confidence_level = 0.95
mu = returns.mean()           # 真实历史均值
sigma = returns.std()         # 真实历史标准差

# ---------------------------------------------------------
# 方法一：参数法 (Parametric VaR)
# 假设收益率服从正态分布。资料中提到的有效边界计算常基于此类正态性假设。
# ---------------------------------------------------------
z_score = norm.ppf(1 - confidence_level)
# 计算日 VaR (金额)
var_parametric = initial_investment * (mu + z_score * sigma)
print(f"参数法 VaR (95% 置信度): {abs(var_parametric):.2f}")

# ---------------------------------------------------------
# 方法二：历史模拟法 (Historical Simulation)
# 利用过去的实际收益率分布来预测未来，不假设分布形状。
# ---------------------------------------------------------
# 模拟过去500天的历史收益率数据
var_historical = initial_investment * np.percentile(returns, (1 - confidence_level) * 100)
print(f"历史模拟法 VaR (95% 置信度): {abs(var_historical):.2f}")

# ---------------------------------------------------------
# 方法三：蒙特卡洛模拟法 (Monte Carlo Simulation)
# 通过大量随机抽样来模拟可能的未来路径。
# ---------------------------------------------------------
simulations = 10000
sim_returns = np.random.normal(mu, sigma, simulations)
m_percentile = np.percentile(sim_returns, (1 - confidence_level) * 100)
var_monte_carlo = initial_investment * m_percentile

print(f"蒙特卡洛法 VaR (95%): ${abs(var_monte_carlo):,.2f}")
```

| **方法**    | **核心假设** | **优点**           | **缺点**                      |
| --------- | -------- | ---------------- | --------------------------- |
| **参数法**   | 正态分布     | 计算极快，公式简单。       | 无法解释“黑天鹅”事件（非正态分布）。         |
| **历史模拟法** | 历史会重演    | 捕捉真实分布特征（肥尾、偏度）。 | 极度依赖样本区间；如果过去两年很平稳，会严重低估风险。 |
| **蒙特卡洛法** | 随机演化     | 可扩展性强，可模拟复杂组合。   | 计算量大；本质上仍受限于你输入的统计模型。       |
