在资本资产定价模型（CAPM）中，计算**风险溢价（Risk Premium）**通常分为两个层面：

**市场风险溢价**（整个市场的超额收益）

**资产风险溢价**（特定股票或组合的超额收益）。

其核心公式如下：

- **市场风险溢价 (Market Risk Premium)** = $E(r_m) - r_f$
    
- **资产风险溢价 (Asset Risk Premium)** = $\beta \times (E(r_m) - r_f)$
    

# 计算公式

$$E(r_i) = r_f + \beta_i (E(r_m) - r_f)$$

其中：

- $r_f$：无风险利率（通常用国债收益率）。
    
- $\beta_i$：资产的贝塔系数，衡量其相对于市场的波动性。
    
- $E(r_m) - r_f$：市场风险溢价。

# 贝塔系数 (Beta)

$$\beta_i = \frac{\text{Cov}(r_i, r_m)}{\text{Var}(r_m)}$$

$\text{Cov}{(r_i，r_m)}$ 资产收益率与市场收益率的**协方差** 

$Var_{r_m}$ 市场收益率的**方差**

# 代码

我们可以使用 `yfinance` 获取数据，并利用 `numpy` 或 `scipy` 计算 Beta，最后得出风险溢价。

```python
import yfinance as yf
import numpy as np

# 1. 下载数据 (以苹果 AAPL 和标普500 ^GSPC 为例)
tickers = ['AAPL', '^GSPC']
data = yf.download(tickers, start="2023-01-01", end="2024-01-01")['Close']

# 2. 计算日收益率
returns = data.pct_change().dropna()

# 3. 设定参数
rf = 0.04  # 假设无风险利率为 4% (年化)
# 将年化无风险利率转换为日化 (简单处理)
daily_rf = rf / 252

# 4. 计算 Beta (资产收益率与市场收益率的协方差 / 市场收益率的方差)
covariance = np.cov(returns['AAPL'], returns['^GSPC'])[0, 1]
market_variance = np.var(returns['^GSPC'])
beta = covariance / market_variance

# 5. 计算市场年化收益率
market_annual_return = returns['^GSPC'].mean() * 252

# 6. 计算风险溢价
market_risk_premium = market_annual_return - rf
asset_risk_premium = beta * market_risk_premium

print(f"Beta 系数: {beta:.2f}")
print(f"市场风险溢价: {market_risk_premium:.2%}")
print(f"该资产(AAPL)的风险溢价: {asset_risk_premium:.2%}")
```

```bash
Beta 系数: 1.11
市场风险溢价: 19.23%
该资产(AAPL)的风险溢价: 21.32%
```

# QA

正数是不是表示未来会亏损?

在CAPM框架下，风险溢价（Risk Premium）是指资产收益率超过无风险利率（如国债收益率）的部分。

- **市场风险溢价 (19.23%)**：表示投资者预期市场整体比无风险资产多赚 19.23%。
- **AAPL风险溢价 (21.32%)**：表示投资者预期AAPL比无风险资产多赚 21.32%。
- **结论**：因为 21.32% 是正数，这意味着模型预测 AAPL 的未来收益会**高于**无风险利率，而不是亏损。

# 数据来源建议

|**变量**|**常用参考**|
|---|---|
|**无风险利率 ($r_f$)**|美国 10 年期国债收益率（常用代码 `^TNX`）。|
|**市场收益率 ($r_m$)**|标普 500 指数 (`^GSPC`) 或 沪深 300 (`000300.SS`)。|
|**收益率频率**|业界常用 **周收益率** 或 **月收益率** 来计算 $\beta$，以过滤日内噪音。|
