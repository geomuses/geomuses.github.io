# yfinance

最易用的历史数据利器

它通过爬虫技术封装了 Yahoo Finance 的数据。
- **功能**：获取股票、基金、汇率的历史 K 线（OHLCV）、拆分合并信息、甚至分析师预期。
- **优势**：无需注册 API Key，一行代码下载数据。
- **局限性**：数据实时性较差（通常延迟 15 分钟），且由于是爬虫封装，偶尔会因为 Yahoo 接口变动而失效。
    
```python
import yfinance as yf
data = yf.download("AAPL", start="2024-01-01", end="2026-04-01")
```

|**参数**|**说明**|**示例**|
|---|---|---|
|**period**|数据范围|`1d, 5d, 1mo, 1y, max`|
|**interval**|时间间隔|`1m, 5m, 1h, 1d, 1wk`|
|**start / end**|指定日期|`YYYY-MM-DD`|
|**actions**|是否下载分红/拆股|`True / False`|



# futu-api

富途牛牛专业交易接口
- **功能**：支持港股、美股、A 股通的**行情获取**与**自动化下单**。
- **优势**：
	- **实时行情**：获取美股、港股、A股（通）的逐笔成交和深度买卖盘。
	- **自动化交易**：直接通过代码下单、撤单、管理持仓
    
```python
from futu import *
quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)
quote_ctx.subscribe(['US.AAPL'], [SubType.QUOTE, SubType.TICK])
```

 # matplotlib

它是 `python` 生态中最基础、功能最强大的绘图库
- **功能**：绘制折线图（K线图的基础）、散点图、直方图等。
- **在金融中的应用**：
    - 绘制股票走势图。
    - 可视化策略的回测收益曲线。
    - 绘制风险分布图
```python
import yfinance as yf 
import matplotlib.pyplot as plt 
data = yf.download('AAPL', start='2023-01-01', end='2024-01-01') 
data['Close'].plot(title="Apple Stock Price") 
plt.show()
```
