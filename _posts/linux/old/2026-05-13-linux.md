# Ubuntu

**Ubuntu**是一款基于 Debian 的开源 Linux 发行版

# 学习这门课程可以学到什么？

学习 Ubuntu（以及 Linux 系统）不仅是掌握一个操作系统，更是进入**后端开发、数据科学和系统架构**大门的钥匙。

# 在微软子系统下载Ubuntu

首先，你需要确保 Windows 的相关功能已开启。

1. 以 **管理员身份** 运行 PowerShell 或命令提示符 (CMD)。
    
2. 输入以下命令并回车：
    
    ```bash
    wsl --install
    ```
    
    _注：这个命令会自动启用必要的组件（虚拟机平台和 WSL 核心），并默认下载最新的 Ubuntu 内核。_
    
3. **重启电脑**。这是必须的操作，以完成组件安装。
