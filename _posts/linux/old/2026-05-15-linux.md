更新本地的“软件索引列表”

```bash
sudo apt update -y
```

根据 `update` 得到的列表，真正开始下载并安装所有可更新的软件包

```bash
sudo apt upgrade -y
```

安装指定的软件包

```bash
sudo apt install <name>.deb
```

卸载软件，但保留其配置文件

```bash
sudo apt remove <package_name>
```

彻底删除。卸载软件的同时，连带配置文件也一并抹除

```bash
sudo apt purge <package_name>
```

清理不再需要的“孤立”依赖包
当你卸载了一个大软件，它之前顺带安装的小插件可能还在占空间，用这个命令一键扫除。

```bash
sudo apt autoremove
```

删除已下载的 `.deb` 安装包缓存

```bash
sudo apt clean
```

在庞大的官方仓库中搜索关键词

```bash
apt search <keyword>
```

查看软件的详细信息，包括版本、大小、开发者描述以及它依赖哪些组件

```bash
apt show <package_name>
```
