在 Linux 中，一切皆文件。

你目前所处的位置（Current Working Directory）决定了你能看到什么。

- **`/` (Root):** 系统的起点。想象成整棵大树的树根。
    
- **`~` (Home):** 你的私人领地。通常是 `/home/你的用户名`。
    
- **`.` (Current):** 当前文件夹。
    
- **`..` (Parent):** 上一级文件夹。
    

# 指令详解

| **指令**    | **用途** | **专家提示**                                |
| --------- | ------ | --------------------------------------- |
| `pwd`     | 我在哪？   | 输出完整路径（Print Working Directory）。        |
| `ls -lah` | 看看有什么  | `-l` 列表模式，`-a` 显示隐藏文件，`-h` 人类可读（KB/MB）。 |
| `cd -`    | 瞬间折返   | 跳回上一个所在的目录，非常适合在两个深层路径间切换。              |
| **`mkdir -p path/to/dir`**| 递归创建。| 即使父目录不存在，它也会帮你一并建好。
| **`cp -r`** | 复制文件夹必备。| `-r` (recursive) 代表递归处理。|
| **`mv`**| 既是移动，也是重命名。本质上都是改变文件的路径。 | 
| **`rm -rf`**|  `-r` 递归，`-f` 强制（不提示）。 |
| **`cat`**| 把内容一股脑倒出来。适合短小的文件。| 
| **`head -n 20`**| 只看前 20 行（默认 10 行）。 |
| **`tail -f`**| 它会挂起并实时显示文件新增的内容。| 常用于观察服务器日志流。
| **`less`**| 翻页神器。| 输入 `less filename` 后，按 `Space` 翻页，按 `q` 退出。
