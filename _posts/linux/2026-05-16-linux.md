`Ctrl` + `b` -> `c`、`%`

|**功能**|**快捷键（先按 Ctrl+b）**|
|---|---|
|**左右**分屏|`%`|
|**上下**分屏|`"`|
|**光标切换**到其他面板|方向键 `↑` `↓` `←` `→` （或 `o` 循环切换）|
|**放大/缩小**当前面板|`z` (Zoom，再次按 `z` 变回原样)|
|**关闭**当前面板|`x` (或者输入 `exit`)|
|显示面板编号|`q`|

# 离开会话（后台挂机 Detach）

假设你在其中一个面板里启动了一个需要运行大半天的程序（比如爬虫脚本或模型训练）。

现在你想关掉电脑下班，或者临时去干别的事，**千万不要直接关终端窗口**，而是要“分离”会话：

- 快捷键：先按 `Ctrl + b` ➔ 松开 ➔ 按 `d` (Disconnect/Detach)

# 控制 tmux

新建 demo

```bash
tmux new -s demo
```

显示 tmux

```bash
tmux ls
```

进入 demo 这个 session

```bash
tmux a -t demo
```

杀死 demo 

```bash
tmux kill-session -t demo
```
