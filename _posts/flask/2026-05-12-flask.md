# 基础配置

首先需要安装扩展并初始化配置。

```python
from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)

# 配置数据库连接 URL (这里以 SQLite 为例)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///example.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False

db = SQLAlchemy(app)

```

# 定义模型 (Models)

模型对应数据库中的表。每个类属性对应一个字段。

```python
class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'

```

# 核心 CRUD 操作

一旦模型定义好了，你就可以通过 `db.session` 执行增删改查。

## **C - 创建 (Create)**

```python
# 创建新对象
new_user = User(username='coding_cat', email='cat@example.com')
# 添加到会话
db.session.add(new_user)
# 提交到数据库
db.session.commit()

```

## **R - 读取 (Read)**

```python
# 获取所有记录
users = User.query.all()

# 按主键查询
user = User.query.get(1)

# 按条件过滤
user = User.query.filter_by(username='coding_cat').first()

```

## **U - 更新 (Update)**

```python
user = User.query.filter_by(username='coding_cat').first()
user.email = 'new_email@example.com'
db.session.commit() # 直接提交即可

```

## **D - 删除 (Delete)**

```python
user = User.query.get(1)
db.session.delete(user)
db.session.commit()

```
