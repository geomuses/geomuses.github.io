# 什么是flask?

Flask 是 Python 的一个 轻量级 Web 框架。
所谓轻量，就是它不强迫你使用特定的开发方式，也不会加入太多你用不到的功能。

- 简单。 你只需要几十行代码就可以启动一个完整的 Web 服务，非常适合初学者。

- 灵活。 Flask 只提供最核心的功能，例如路由和请求处理，其他功能你可以自由选择套件来扩充，想怎么写就怎么写。

- 可扩展。 Flask 的生态系统非常丰富，像数据库操作、用户登录、表单验证、REST API 设计都有对应的扩展可以使用。

 ## 1. 环境搭建与路由（Routing）

这是网站的地图，决定了用户访问不同网址时会看到什么。

- **虚拟环境**：学会创建独立的开发空间，让项目依赖井然有序。
    
- **路由配置**：定义 URL 路径，利用装饰器把网址和 Python 函数精准绑定。

 ## 2. 处理请求与响应（Request & Response）

这是服务器与用户之间的对话方式。

- **获取输入**：通过 `request` 对象接收用户提交的表单或搜索关键词。
    
- **逻辑反馈**：根据用户动作，利用 `redirect`（重定向）或 `render_template` 给出正确的反馈。

 ## 3. 动态模板渲染（Jinja2 Rendering）

让网页不再是死板的静态文件，而是具有灵魂。

- **数据注入**：将 Python 后台处理好的变量、列表、字典直接显示在前端 HTML 中。
    
- **模板继承**：像写模块一样复用代码，一次编写导航栏，全站通用。

 ## 4. 静态资源与样式美化（Static Files）

让你的 Web 应用从“简陋文本”进化为“专业网站”。

- **文件管理**：系统地引入 CSS 美化布局，使用 JavaScript 增加交互，并加载图片资源。
    
- **前端整合**：学会如何在 Flask 框架下正确引用外部样式库，提升视觉体验。

## 5. 错误处理与日志监控（Error Handling）

确保你的程序在面对异常时依然优雅。

- **异常拦截**：自定义 404（未找到）和 500（服务器错误）页面，提升用户体验。
    
- **运行监控**：通过日志（Logging）记录系统运行状态，像侦探一样快速定位并修复 Bug。

# 虚拟环境

Python 虚拟环境是一个隔离的 Python 运行环境，让你在同一台电脑上安装和使用多个项目所需的不同版本的 Python 包，而不会相互干扰。

使用 venv 创建和管理虚拟环境

以下以 venv 为例：

创建虚拟环境

```bash
python -m venv env
```

env 是虚拟环境的目录名，可以自定义

会创建一个名为 env/ 的文件夹，里面包含独立的 Python 解释器和 site-packages 包目录

激活虚拟环境

Windows：

```bash
.\env\Scripts\activate
```

macOS / Linux：

```bash
source env/bin/activate
```

退出虚拟环境

```bash
deactivate
```
