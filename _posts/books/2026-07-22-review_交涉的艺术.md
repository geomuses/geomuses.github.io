// 设计plan b获取稳定性
有能力通过完全竞争来舍得地方放弃权益?
是否有策略使得对方投降基于另外一个组织的压价(也是对方,通过价格舍得对方投降)

1.我該交涉談判嗎?
怕被讨厌,當進一步的要求所產生的風險比可能獲得的好處還大的時候,就該停止談判。
2.現在是交涉談判的好時機嗎?
時間是否站在你這邊,取決於你的「談判協議最佳替代方案」的穩定性

3.我該把籌碼全部押下去,還是要採取避險動作?
不用。
許多談判並不需要做出籌碼全押的動作,只需要試探水溫就行了

考虑坏情况 // 拟定假设环境

展開談判的時候,你要暫時性地辨識出一個讓你很難決定該接受或拒絕對方提出最終方案的交易點。這麼做可以
檢驗你的決心,也為你提供一個評估替代方案的標準根據。

只是在劃一條你的拒絕線,還沒有訂出任何目標

对方底线 // 可损失的框架 >对方不想要的损失> 对方的要求? 在这个框架上自身获得的好处。 暂时一篇空白
最好底线 // 希望可以达到的目标
弹性底线 在交涉策略上的可控制 // 自身需要的内容
+一个助理协助策略的完整性和结构性(外在限制)

1.研擬備案計畫。協商過程中的障礙可能隱藏著另一個不同交易的種子,甚至可能比原先期望的交易更好。
2.使用多種話術。帶著蘿蔔與棍棒。
3.保護你的退場選擇。就像玩撲克牌一樣,知道何時該留著牌,何時該亮牌;何時該離開,何時該前進。
